# liri-node-app
Language Interpretation and Recognition Interface

#Usage
## Get Tweets
```node liri my-tweets```

## Search Song
```node liri spotify-my-song <'song name'>```

## Search Movie
```node liri movie-this <'movie name'>```

## Read from file & Run
### random.txt Examples
* ```my-tweets```
* ```movie-this,<'movie name'>```
* ```spotify-my-song, <'song name'>```
### Run:
```node liri do-what-it-says```
